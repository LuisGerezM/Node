### Envio por Email
  - Plano
  - HTML
  - Con attachments

# dev
1. Clonar el archivo env.template a .env
2. Configurar las variables de entorno
3. Ejecutar el comando ``` npm install ```
4. Levantar BD con el comando
  ```
    docker compose up -d

  ```
5. Ejecutar ``` npm run dev ```
